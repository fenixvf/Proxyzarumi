zarumi-drivea/worker.js — linha const WORKER_BASE
zarumi-aq/worker.js — linha const WORKER_BASE
zarumi-at/worker.js — linha const WORKER_BASE
zarumi-proxy/worker.sw.js — nas 3 URLs dos extratores
3. Deploy (via Cloudflare Dashboard)
